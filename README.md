# Balanced

Online Marketplace Payments

## Installation

Add this line to your application's Gemfile:

    gem 'balanced'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install balanced

## Usage

See https://www.balancedpayments.com/docs/ruby for tutorials and documentation.
